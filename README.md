# 项目介绍
## 感谢DeltaF的教学，追随DeltaF学习量化投资。
本项目致力于研究股票、基金等标的的投资量化，项目基于Delta-F的项目DeltaTrader，项目地址：https://github.com/Delta-F/DeltaTrader
## 一、 数据来源
1. 感谢[tushare地址提供的数据](https://tushare.pro/register?reg=400208)
